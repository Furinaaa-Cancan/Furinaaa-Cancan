### Bioinformatics

目前主要做基于大型人群队列数据库的生信分析，常用数据源包括 NHANES、KNHANES、UK Biobank、CHARLS、ELSA、ELSA-Brasil 等，核心方法是机器学习预测建模与无监督聚类，用来挖掘健康相关的风险因素与表型分群。
