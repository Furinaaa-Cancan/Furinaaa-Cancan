### Bioinformatics

**Cohort analysis:** bioinformatics analyses based on large population cohort databases — **NHANES · KNHANES · UK Biobank · CHARLS · ELSA · ELSA-Brasil**. Building a **governance framework for AI-driven clinical prediction models** under top-journal review standards.

**Nutrition:** **LLM-assisted processing** of nutrition databases — **FNDDS · PE Polyphenol · USDA**.
